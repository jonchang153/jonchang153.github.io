## Create the plot
We'll import matplotlib to create the visualization. Create a figure with one axes using the `subplots()` function.


```python
from matplotlib import pyplot as plt
fig, ax = plt.subplots(1)
```


    

    


Next, to visualize relationships between data of two sets, use a function such as `scatter()` or `plot()`. The `hist()` or `boxplot()` functions can be used instead for the data distribution of a single set. Pass in the data you want to visualize by specifying the column name of the DataFrame, `penguins['column_name']`. 


```python
# scatter plot of the columns 'Culmen Length (mm)' and 'Culmen Depth (mm)' of penguins
# the first argument plotted along the x-axis and the second argument plotted along the y-axis.
ax.scatter(penguins['Culmen Length (mm)'], penguins['Culmen Depth (mm)'])
```








## Customize the plot
Finally, you can set the title, x-axis label, and y-axis label of the plot.


```python
ax.set(title = 'Culmen Depth vs Culmen Length',
       xlabel = 'Culmen Length (mm)',
       ylabel = 'Culmen Depth (mm)')
```








Display the plot:


```python
fig
```

## Additional features
You can create multiple axes, change the size of the figure, and add a label/legend if plotting multiple sets of points. You can also style each set of points by changing features such as color, opacity, marker type, line width, etc.

For something more advanced, play around with the `groupby()` and corresponding `apply()` functions. These allow you to separate the data into groups based on a specified column of the DataFrame, and plot each group's data separately by applying a helper function (allowing each group to have different stylings).
